CAUB - Chrome OS WiFi ONC Builder

Use Chrome 85+ to convert favorite WiFi details into an importable ONC file.

CAUB pulls WiFi metadata from chrome://network#state so you can generate an ONC file for importing preferred networks.

Note: Expand each “Favorite” WiFi entry so the details are visible before copying.

Instructions:

  1. Go to chrome://network#state
  2. Scroll to the “Favorite” WiFi section
  3. Click the + icon for each WiFi entry to expand details
  4. Press Ctrl + A, then Ctrl + C to copy all text
  5. Paste the copied text into the field below
  6. Click “Generate Configuration File”

Alternative method: Legacy version

Next Steps:

  1. Download the generated ONC file
  2. Open chrome://network#general
  3. Click “Import ONC” and upload the file
  4. Verify the preferred networks were added

Educational Use: This tool is designed for educational purposes to help students understand network configuration principles.
