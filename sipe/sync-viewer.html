SIPE - Sync Internals Password Extractor

Decode synced WiFi configuration details from chrome://sync-internals.

SIPE reads WiFi sync entries (wifi_) so you can inspect network configuration details from synced devices.

Results will appear here...

Instructions:

  1. Go to chrome://sync-internals and open the “Search” tab
  2. Search for wifi_ and click the network you want
  3. Copy all of the JSON shown in the details pane
  4. Paste it above and click “Analyze Data”

Note: This tool works with synchronized network data from Chrome-based devices. For managed networks, additional configuration may be required.
